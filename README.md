# rgn-api
API for the RGN website.

## Setup

* Install rethinkdb

```
brew install rethinkdb
```

* Start database

```
rethinkdb
```

* Install dependencies

```
npm install
```

* Run app

```
npm start
```

## Testing

```
npm test
```
